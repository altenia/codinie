Programming is should be fun!

Programming is (usually) fun. Programming becomes tedious when coding no-brainer boiler plate parts such as model or DAO clases where you have to repeat attributes of the schema entities over and over. Most of it can usually be (and should be) automated. Here is where Codini comes to play. Codini is a PHP based code generator. It generates codes based on database metadata, or other schema definition such as Hibernate's HBM file.

Codini Highlights:
  • Uses customizable templating (PHP based) to generate code.
  • Multiple projects.
  • Supports multiple data sources: MySQL, Hibernate, MS SQL.
  • Pre-built templates for Java and Scala Model and DAO .
